## api 구현 기능
<user>
- 회원가입: signUp <br/>
- 회원탈퇴: signOut <br/>
- 로그인: login <br/>
- 내 id 조회: me <br/>

<room>
방 목록 조회: rooms <br/>
방 상세페이지 조회: room <br/>
방 즐겨찾기: likeRoom <br/>

<reservation>
방 예약하기: reservationRoom <br/>
