## api 구현 기능
<user>
회원가입 - signUp
회원탈퇴 - signOut 
로그인 - login
내 id 조회 - me

<room>
방 목록 조회 - rooms
방 상세페이지 조회 - room
방 즐겨찾기 - likeRoom

<reservation>
방 예약하기 - reservationRoom
